Wuthering Waves:

A game that has shown more improvement than many others.

-- Wuthering Waves is a game set in an apocalyptic world where you fight against monsters of different kind. It is an open world game like Genshin Impact with a gacha system as well to get the characters you want.

5 reasons why this game is my favourite.

  • Combat is it's main strength, it's very fluid and fun to learn.
  • Graphics are incredible, you can truly see it's beauty when you have a high-end pc.
  • Developers are constantly adding and improving content so that there is something to do.
  • The game has been fun even when there aren't any new content yet.
  • Because of the combat, it can make the game challenging yet rewarding when you complete it.
